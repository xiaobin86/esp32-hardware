## 一、开发板：ESP32-S3-DevKitC-1

- **型号**：ESP32-S3-DevKitC-1-N16R8
- **Flash**：16MB
- **PSRAM**：8MB
- **工作电压**：3.3V（USB供电5V）

## 三、详细接线图

### 3.1 ST7789V 显示屏（10针带字库版）

**物理布局**（上下两排，共10针）：
```
上排: FSO    CS1    RES    SCL    GND
下排: CS2    BLK    DC     SDA    VCC
```

**接线表**：

| 显示屏 | 接ESP32-S3 | J1位置 | 功能 | 线色 |
|--------|-----------|--------|------|------|
| VCC | 3.3V | J1-1 | 电源 | 红色 |
| GND | GND | J1-22 | 地线 | 黑色 |
| BLK | GPIO5 | J1-5 | 背光控制 | 黄色 |
| DC | GPIO6 | J1-6 | 数据/命令选择 | 白色 |
| RES | GPIO7 | J1-7 | 复位 | 灰色 |
| CS2 | GPIO9 | J1-15 | 字库芯片片选 | 棕色 |
| CS1 | GPIO10 | J1-16 | 显示屏片选 | 黑色 |
| SDA | GPIO11 | J1-17 | SPI MOSI数据 | 红色 |
| SCL | GPIO12 | J1-18 | SPI SCK时钟 | 青色 |
| FSO | GPIO13 | J1-19 | SPI MISO字库回读 | 粉色 |

**接线示意图**：
```
ST7789显示屏                ESP32-S3 J1
┌─────────────┐            ┌──────────┐
│ VCC  [红]   ├────────────┤ J1-1 3V3 │
│ GND  [黑]   ├────────────┤ J1-22 GND│
│ BLK  [黄]   ├────────────┤ J1-5 GPIO5│
│ DC   [白]   ├────────────┤ J1-6 GPIO6│
│ RES  [灰]   ├────────────┤ J1-7 GPIO7│
│ CS2  [棕]   ├────────────┤ J1-15 GPIO9│
│ CS1  [黑]   ├────────────┤ J1-16 GPIO10│
│ SDA  [红]   ├────────────┤ J1-17 GPIO11│
│ SCL  [青]   ├────────────┤ J1-18 GPIO12│
│ FSO  [粉]   ├────────────┤ J1-19 GPIO13│
└─────────────┘            └──────────┘
```

**关键注意事项**：
- SCL/SDA 为屏幕和字库共用，通过CS1/CS2分时片选
- 字库芯片型号：GT30L32S4W（GB2312编码，16×16点阵）
- 代码文件必须以 **GB2312/ANSI编码** 保存才能正确显示中文
- `tft.init()` 会改变SPI配置，读取字库前需重新初始化SPI

### 3.2 GY-91 传感器（MPU6500 + BMP280）

**物理布局**（一排8针）：
```
VIN    3V3    GND    SCL    SDA    SAO/SDO    NCS    CSB
```

**接线表（I2C模式）**：

| GY-91 | 接ESP32-S3 | J1位置 | 功能 | 线色 |
|-------|-----------|--------|------|------|
| VIN | 5V | J1-21 | 电源输入（经板载LDO稳压） | 红色 |
| 3V3 | NC悬空 | — | 3.3V输出（可为其他设备供电） | — |
| GND | GND | — | 地线 | 黑色 |
| SCL | GPIO4 | J1-4 | I2C时钟 | 绿色 |
| SDA | GPIO3 | J1-13 | I2C数据 | 蓝色 |
| SAO/SDO | GND | — | 地址选择（GND=0x68/0x76） | 黑色 |
| NCS | NC悬空 | — | SPI片选（I2C模式不接） | — |
| CSB | NC悬空 | — | SPI片选（I2C模式不接） | — |

**接线示意图**：
```
GY-91传感器                 ESP32-S3 J1
┌─────────────┐            ┌──────────┐
│ VIN  [红]   ├────────────┤ J1-21 5V │
│ GND  [黑]   ├────────────┤ J1-22 GND│
│ SCL  [绿]   ├────────────┤ J1-4 GPIO4│
│ SDA  [蓝]   ├────────────┤ J1-13 GPIO3│
│ SAO  [黑]   ├────────────┤ GND      │
└─────────────┘            └──────────┘
```

**关键参数**：
- I2C地址：MPU-6500 = `0x68`，BMP280 = `0x76`
- WHO_AM_I：MPU-6500 = `0x70`
- 实际自由度：7 DOF（加速度3轴 + 陀螺仪3轴 + 气压计1轴）
- 注意：该模块**无磁力计**，无法实现9轴姿态解算

### 3.3 MAX98357A 功放模块

**物理布局**（一排7针）：
```
LRC    BCLK    DIN    GAIN    SD    GND    VIN
```

**接线表**：

| MAX98357A | 接ESP32-S3 | J3位置 | 功能 | 线色 |
|-----------|-----------|--------|------|------|
| LRC | GPIO21 | J3-18 | I2S声道时钟（WS） | 银色 |
| BCLK | GPIO47 | J3-17 | I2S位时钟 | 金色 |
| DIN | GPIO1 | J3-4 | I2S数据输入 | 橙色 |
| GAIN | 悬空 | — | 增益控制（悬空=9dB） | — |
| SD | 悬空 | — | 关断控制（悬空=常开） | — |
| GND | GND | — | 地线 | 黑色 |
| VIN | 3.3V/5V | — | 供电（5V音量更大） | 红色 |

**接线示意图**：
```
MAX98357A功放              ESP32-S3 J3
┌─────────────┐            ┌──────────┐
│ LRC  [银]   ├────────────┤ J3-18 GPIO21│
│ BCLK [金]   ├────────────┤ J3-17 GPIO47│
│ DIN  [橙]   ├────────────┤ J3-4 GPIO1│
│ GND  [黑]   ├────────────┤ GND      │

│ OUT-        ├────────────┤ 喇叭-    │
└─────────────┘            └──────────┘
```

**增益配置**：
| 接法 | 增益 |
|------|------|
| 悬空（默认） | 9 dB |
| 接GND | 6 dB |
| 接3V3 | 12 dB |

### 3.4 INMP441 I2S 麦克风

**物理布局**（两排6针）：
```
上排: SD    VDD    GND
下排: L/R   WS     SCK
```

**接线表**：

| INMP441 | 接ESP32-S3 | J3位置 | 功能 | 线色 |
|---------|-----------|--------|------|------|
| VDD | 3.3V | — | 供电 | 红色 |
| GND | GND | — | 地线 | 黑色 |
| SCK | GPIO47 | J3-17 | I2S位时钟（与功放共用） | 金色 |
| WS | GPIO21 | J3-18 | I2S声道时钟（与功放共用） | 银色 |
| SD | GPIO2 | J3-5 | I2S数据输出 | 紫色 |
| L/R | GND | — | 声道选择（GND=左声道） | 黑色 |

**接线示意图**：
```
INMP441麦克风              ESP32-S3 J3
┌─────────────┐            ┌──────────┐
│ VDD  [红]   ├────────────┤ 3.3V     │
│ GND  [黑]   ├────────────┤ GND      │
│ SCK  [金]   ├────────────┤ J3-17 GPIO47│
│ WS   [银]   ├────────────┤ J3-18 GPIO21│
│ SD   [紫]   ├────────────┤ J3-5 GPIO2│
│ L/R  [黑]   ├────────────┤ GND      │
└─────────────┘            └──────────┘
```

**关键特性**：
- 与MAX98357A共用BCLK/WS，ESP32-S3 I2S全双工同时支持输入和输出
- SD是麦克风数据输出引脚，需单独占用GPIO2
- 无需外置ADC，直接输出数字I2S信号

## 六、接线颜色惯例

| 颜色 | 用途 | 说明 |
|------|------|------|
| 红色 | 电源（VCC/VIN/3.3V/5V） | 国际通用正极标识 |
| 黑色 | 接地（GND） | 国际通用负极标识 |
| 绿色 | I2C SCL | 时钟信号 |
| 蓝色 | I2C SDA | 数据信号 |
| 黄色 | 控制信号（BLK/背光） | 易识别 |
| 白色/灰色 | 控制信号（DC/RES） | 中性色 |
| 棕色/黑色 | 片选（CS1/CS2） | 低电平有效 |
| 红色 | SPI MOSI/SDA | 数据输出 |
| 青色/浅蓝 | SPI SCK/SCL | 时钟信号 |
| 粉色 | SPI MISO/FSO | 数据输入 |
| 橙色 | I2S DIN | 音频数据输入 |
| 紫色 | I2S SD | 音频数据输出 |
| 银色/金色 | I2S BCLK/WS | 音频时钟 |
